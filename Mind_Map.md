# 前面不能加副词的单词
- There
- Here
- Home

# Die
- death n.
- dead adj.
- died 过去时

# Make sb
- do sth 使某人做某事
- adj. 使某人怎样

# Choose
- to do sth
- sb/sth

# Finish
- sth
- doing sth

# Hope
- to do sth 希望做某事
- sb + to do sth 希望某人做某事
- 句子

# Seem
- It seem + that... 看来
- Seem to do sth. 似乎做某事
- Semm (to be) + adj./n. ... 看上去像...

# 名词复数
1. 一般+`s`
2. `s`,`x`,`ch`,`sh` + `es`
3. 辅音字母 + `y`结尾 -> 改`y`为`i`加`es`
4. `o`结尾：
	- 有生命 + `es`
	- 无生命 + `s`
